# 🎵 MusicHub

## EN 🇺🇸

**MusicHub** is a modern desktop multimedia application for streaming, downloading, and managing music and videos with high performance and a smooth user experience.

---

## 🚀 Features

### 🎧 Playback

* Audio playback (default mode)
* Video playback (switchable from search page)
* Seamless switching between audio and video modes

### 📝 Smart Lyrics (LRC System)

* Automatic LRC fetching
* Local caching for offline usage
* Click any lyric line → jump to that moment in the track
* Real-time synchronization with playback
* Smooth auto-scrolling and highlighting

### 📂 Media Library

* Select any folder containing music/videos
* Automatic updates using file system monitoring:

  * Detect new files
  * Detect deleted files
  * Detect renamed files
* No need for manual refresh

### 📥 Download System

* Download individual videos
* Playlist download support:

  * Paste playlist URL
  * Preview all videos
  * Select specific videos or download all
* Queue-based download manager

---

## ⚙️ Technical Highlights

* Asynchronous architecture (non-blocking UI)
* Local caching system for performance
* File system watcher integration
* Modular and scalable design

---

## 🖥️ Installation

```bash
git clone https://github.com/Deep-Black-Stone/MusicHub.git
cd MusicHub
```

Install dependencies (depending on your stack):

```bash
# example
pip install -r requirements.txt
# or
npm install
```

Run the application:

```bash
# example
python main.py
# or
npm start
```

* تخزين محلي للعمل بدون إنترنت
